# Manual do Aplicativo

## Instalação
1. Clone o repositório.
2. Execute `npm install`.
3. Execute `npm run dev`.

## Operação
- **Início**: Insira seu nome.
- **Escala**: Use o calendário para atribuir plantões ou a aba "Período" para preenchimento em lote.
- **Configuração**: Ajuste os dados da área para a exportação oficial.

## Teste
- Execute `npm run lint` para verificar a integridade do código.

## Configuração
- Os dados são persistidos via `localStorage`.

## Deleção
- Para limpar os dados, use o botão "Limpar Escala Mensal" ou limpe o cache do navegador.
